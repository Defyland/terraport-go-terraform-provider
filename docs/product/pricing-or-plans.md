# Pricing or Plans

Terraport is a developer tool, not a billable SaaS product. A real platform team would measure value through avoided manual onboarding time, fewer drift incidents, and faster incident recovery.

## Internal Cost Model

- Engineering cost: provider maintenance, schema migrations, and acceptance tests.
- Operational cost: Terraform state storage, access control, and support for drift/import incidents.
- API cost: additional read calls during Terraform refresh and retries during remote rate limits.
